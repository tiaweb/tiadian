Animation

<p class=“tiadian-text-lead”>A collection of smooth animations to use within your page.</p>

Usage

Add one of the .tiadian-animation-* classes to any element. The animation is shown when the class is added, so usually immediately on page load. To show the animation at another point, for example when the element enters the viewport, you would add the class using JavaScript — with the Scrollspy component for instance. This is what happens in many of TIAdian’s components that make use of animations. All animations themselves are implemented with CSS, so they do not require JavaScript to play.

Class Description
.tiadian-animation-fade The element fades in.
.tiadian-animation-scale-up<br> .tiadian-animation-scale-down The element fades in and scales up or down.
.tiadian-animation-slide-top<br> .tiadian-animation-slide-bottom .tiadian-animation-slide-left<br> .tiadian-animation-slide-right The element fades and slides in from the top, bottom, left or right by its own height or width.
.tiadian-animation-slide-top-small<br> .tiadian-animation-slide-bottom-small .tiadian-animation-slide-left-small<br> .tiadian-animation-slide-right-small The element fades and slides in from the top, bottom, left or right with a smaller distance which is specified by a fixed pixel value.
.tiadian-animation-slide-top-medium<br> .tiadian-animation-slide-bottom-medium .tiadian-animation-slide-left-medium<br> .tiadian-animation-slide-right-medium The element fades and slides in from the top, bottom, left or right with a medium distance which is specified by a fixed pixel value.
.tiadian-animation-kenburns The element scales very slowly up without fading in.
.tiadian-animation-shake The element shakes.
.tiadian-animation-stroke The SVG element strokes are drawn.

To toggle an animation on hover or focus, add the .tiadian-animation-toggle class to a parent element. Also add tabindex="0" to make the animation focusable through keyboard navigation and on touch devices.

Ken Burns

To add a simple Ken Burns effect, add the .tiadian-animation-kenburns class to any image. You can also apply the .tiadian-animation-reverse or one of the .tiadian-transform-origin-* classes from the Utility component to modify the effect.

<img class="tiadian-animation-kenburns" src="" alt="">

By default the animation starts on page load. In this example we used the Scrollspy component, to toggle the effect when the image enters the view.

<div class="tiadian-child-width-1-2@s tiadian-grid-small" tiadian-grid> <div> <div class="tiadian-overflow-hidden"> <img src="images/dark.jpg" alt="Example image" tiadian-scrollspy="cls: tiadian-animation-kenburns; repeat: true"> </div> </div> <div> <div class="tiadian-overflow-hidden"> <img src="images/dark.jpg" alt="Example image" class="tiadian-animation-reverse tiadian-transform-origin-top-right" tiadian-scrollspy="cls: tiadian-animation-kenburns; repeat: true"> </div> </div> </div>

SVG Strokes

The Animation component can be used to animate SVG strokes. The effect looks like the SVG strokes are drawn before your eyes. The SVG image has to be injected into the markup as an inline SVG. This can be done manually or using the SVG component.

The following example shows how to add the inline SVG manually. Since you have to know the exact length of the stroke, TIAdian requires you to set the length in the custom property --tiadian-animation-stroke. In this example the stroke length is 46.

<svg class="tiadian-animation-stroke" style="--tiadian-animation-stroke: 46;" viewBox="0 0 20 20" xmlns="http://www.w3.org/2000/svg"> <path fill="none" stroke="#000" stroke-width="1" d=""/> </svg>

A much easier way is to use the SVG component by adding tiadian-svg="stroke-animation: true" to the image element. It will calculate the stroke length and add the --tiadian-animation-stroke custom property automatically.

<img src="" tiadian-svg="stroke-animation: true">
