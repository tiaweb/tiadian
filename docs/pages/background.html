Background

<p class=“tiadian-text-lead”>A collection of utility classes to add different backgrounds to elements.</p>

Usage

To apply a background color to an element, add one of the following classes. The actual color for each modifier is defined by the TIAWEB style that you have chosen or customized.

Class Description
.tiadian-background-default Applies the default background color.
.tiadian-background-muted Applies a muted background color.
.tiadian-background-primary Applies the primary background color.
.tiadian-background-secondary Applies a secondary background color. 
<div class="tiadian-background-primary"></div>

Note To adapt your content for better visibility on each background, add the .tiadian-light or .tiadian-dark class from the Inverse component. Use the Padding component to add some padding to the elements.

Size modifiers

This component features classes to specify the size of background images by keeping its intrinsic ratio.

Class Description
.tiadian-background-cover Scales the background image to completely cover the containing area.
.tiadian-background-contain Scales the background image as far as its width and height can fit inside the containing area.

Note When using these classes, the background position automatically shifts to the middle and background-repeat is set to no-repeat.

Position modifiers

To alter the background position of your image — which is in the top left hand corner of the area by default, add one of the following classes.

Class Description
.tiadian-background-top-left The initial position of the image is in the top left hand corner.
.tiadian-background-top-center The initial position of the image is at the top.
.tiadian-background-top-right The initial position of the image is in the top right hand corner.
.tiadian-background-center-left The initial position of the image is on the left.
.tiadian-background-center-center The initial position of the image is in the middle.
.tiadian-background-center-right The initial position of the image is on the right.
.tiadian-background-bottom-left The initial position of the image is in the bottom left hand corner.
.tiadian-background-bottom-center The initial position of the image is at the bottom.
.tiadian-background-bottom-right The initial position of the image is in the bottom right hand corner.

Responsive

Add one of the following classes to limit the display of background images to certain viewport sizes. This is great in cases where the image and content overlap on a phone screen in a way that would make text illegible.

Class Description
.tiadian-background-image@s Displays the background image on device widths of 640px and larger.
.tiadian-background-image@m Displays the background image on device widths of 960px and larger.
.tiadian-background-image@l Displays the background image on device widths of 1200px and larger.
.tiadian-background-image@xl Displays the background image on device widths of 1600px and larger. 
<div class="tiadian-background-image@m"></div>

Blend modes

Add one of the following classes to apply different blend modes to your background image. You can combine these with the background color classes, as well. For a better understanding of how background blend modes work, take a look at this CSS Tricks article.

Class Description
.tiadian-background-blend-multiply This class sets the blend mode to multiply.
.tiadian-background-blend-screen This class sets the blend mode to screen.
.tiadian-background-blend-overlay This class sets the blend mode to overlay.
.tiadian-background-blend-darken This class sets the blend mode to darken.
.tiadian-background-blend-lighten This class sets the blend mode to lighten.
.tiadian-background-blend-color-dodge This class sets the blend mode to color dodge.
.tiadian-background-blend-color-burn This class sets the blend mode to color burn.
.tiadian-background-blend-hard-light This class sets the blend mode to hard light.
.tiadian-background-blend-soft-light This class sets the blend mode to soft light.
.tiadian-background-blend-difference This class sets the blend mode to difference.
.tiadian-background-blend-exclusion This class sets the blend mode to exclusion.
.tiadian-background-blend-hue This class sets the blend mode to hue.
.tiadian-background-blend-saturation This class sets the blend mode to saturation.
.tiadian-background-blend-color This class sets the blend mode to color.
.tiadian-background-blend-luminosity This class sets the blend mode to luminosity.
