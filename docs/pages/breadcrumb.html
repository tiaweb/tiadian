Breadcrumb

<p class=“tiadian-text-lead”>Create breadcrumbs to show users their location within a website.</p>

Usage

The Breadcrumb component consists of links, which are aligned side by side and separated by a divider. Add the .tiadian-breadcrumb class to a <ul> element to define the component. Use <a> elements as breadcrumb items within the list. An active state is automatically applied to the last <li> element.

To add list items without a link, use a <span> element instead of an <a>. Alternatively, disable an <a> element by adding the .tiadian-disabled class to the <li> element and remove the href attribute from the anchor to make it inaccessible through keyboard navigation.

<ul class="tiadian-breadcrumb">
    <li><a href=""></a></li>
    <li><a href=""></a></li>
    <li><span></span></li>
</ul>
