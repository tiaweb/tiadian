Button

<p class=“tiadian-text-lead”>Easily create nice looking buttons, which come in different styles.</p>

Usage

To apply this component, add the .tiadian-button class and a modifier such as .tiadian-button-default to an <a> or <button> element. Add the disabled attribute to a <button> element to disable the button.


<a class="tiadian-button tiadian-button-default" href=""></a>

<button class="tiadian-button tiadian-button-default"></button>

<button class="tiadian-button tiadian-button-default" disabled></button>

Link 


<p tiadian-margin>
    <a class="tiadian-button tiadian-button-default" href="#">Link</a>
    <button class="tiadian-button tiadian-button-default">Button</button>
    <button class="tiadian-button tiadian-button-default" disabled>Disabled</button>
</p>

Note If you are displaying a number of buttons in a row, you can add a top margin to them, when they stack on smaller viewports. Just add the tiadian-margin attribute from the Margin component to their parent element.

Style modifiers

There are several style modifiers available. Just add one of the following classes to apply a different look.

Class Description
.tiadian-button-default Default button style.
.tiadian-button-primary Indicates the primary action.
.tiadian-button-secondary Indicates an important action.
.tiadian-button-danger Indicates a dangerous or negative action.
.tiadian-button-text Applies an alternative, typographic style.
.tiadian-button-link Makes a <button> look like an <a> element. 

<button class="tiadian-button tiadian-button-primary"></button>



<p tiadian-margin>
    <button class="tiadian-button tiadian-button-default">Default</button>
    <button class="tiadian-button tiadian-button-primary">Primary</button>
    <button class="tiadian-button tiadian-button-secondary">Secondary</button>
    <button class="tiadian-button tiadian-button-danger">Danger</button>
    <button class="tiadian-button tiadian-button-text">Text</button>
    <button class="tiadian-button tiadian-button-link">Link</button>
</p>

Size modifiers

Add the .tiadian-button-small or .tiadian-button-large class to a button to make it smaller or larger.


<button class="tiadian-button tiadian-button-default tiadian-button-small"></button>

<button class="tiadian-button tiadian-button-default tiadian-button-large"></button>



<p tiadian-margin>
    <button class="tiadian-button tiadian-button-default tiadian-button-small">Small button</button>
    <button class="tiadian-button tiadian-button-primary tiadian-button-small">Small button</button>
    <button class="tiadian-button tiadian-button-secondary tiadian-button-small">Small button</button>
</p>

<p tiadian-margin>
    <button class="tiadian-button tiadian-button-default tiadian-button-large">Large button</button>
    <button class="tiadian-button tiadian-button-primary tiadian-button-large">Large button</button>
    <button class="tiadian-button tiadian-button-secondary tiadian-button-large">Large button</button>
</p>

Width modifiers

Add the .tiadian-width-1-1 class from the Width component and the button will take up full width.

Example


<button class="tiadian-button tiadian-button-default tiadian-width-1-1 tiadian-margin-small-bottom">Button</button>
<button class="tiadian-button tiadian-button-primary tiadian-width-1-1 tiadian-margin-small-bottom">Button</button>
<button class="tiadian-button tiadian-button-secondary tiadian-width-1-1">Button</button>

Group

To create a button group, add the .tiadian-button-group class to a <div> element around the buttons. That’s it! No further markup needed.


<div class="tiadian-button-group">
    <button class="tiadian-button tiadian-button-default"></button>
    <button class="tiadian-button tiadian-button-default"></button>
    <button class="tiadian-button tiadian-button-default"></button>
</div>


<div>
    <div class="tiadian-button-group">
        <button class="tiadian-button tiadian-button-secondary">Button</button>
        <button class="tiadian-button tiadian-button-secondary">Button</button>
        <button class="tiadian-button tiadian-button-secondary">Button</button>
    </div>
</div>

<div class="tiadian-margin-small">

Button group with dropdowns

Use button groups to split buttons into a standard action on the left and a dropdown toggle on the right. Just wrap the toggling button and the drop or dropdown inside a <div> element and add the .tiadian-inline class from the Utility component.


<!-- A button group with a dropdown -->
<div class="tiadian-button-group">
    <button class="tiadian-button tiadian-button-default"></button>
    <div class="tiadian-inline">

        <!-- The button toggling the dropdown -->
        <button class="tiadian-button tiadian-button-default" type="button"></button>
        <div tiadian-dropdown="mode: click; boundary: ! .tiadian-button-group; boundary-align: true;"></div>

    </div>
</div>
