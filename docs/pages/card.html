Card

<p class=“tiadian-text-lead”>Create layout boxes with different styles.</p>

Usage

The Card component consists of the card itself, the card body and an optional card title. Typically, cards are arranged in grid columns from the Grid component.

Class Description
.tiadian-card Add this class to a <div> element to define the Card component.
.tiadian-card-body Add this class to the card to create padding between the card and its content.
.tiadian-card-title Add this class to a heading to define a card title. 

<div class="tiadian-card tiadian-card-body">
    <h3 class="tiadian-card-title"></h3>
</div>

By default, a card is blank. That is why it is important to add a modifier class for styling. In our example we are using the .tiadian-card-default class.

Default

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.


<div class="tiadian-card tiadian-card-default tiadian-card-body tiadian-width-1-2@m">
    <h3 class="tiadian-card-title">Default</h3>
    <p>Lorem ipsum <a href="#">dolor</a> sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
</div>

Header & footer

You can also divide a card into header and footer — around the default body. Just add the .tiadian-card-header or .tiadian-card-footer class to a <div> element inside the card.


<div class="tiadian-card">
    <div class="tiadian-card-header">
        <h3 class="tiadian-card-title"></h3>
    </div>
    <div class="tiadian-card-body"></div>
    <div class="tiadian-card-footer"></div>
</div>

Media

To display an image inside a card without any spacing, add one of the following classes to a <div> around the <img> element. Mind that you need to modify the source order accordingly.

Class Description
.tiadian-card-media-top This class indicates that the media element is aligned to the top.
.tiadian-card-media-bottom This class indicates that the media element is aligned to the bottom.
.tiadian-card-media-left This class indicates that the media element is aligned to the left.
.tiadian-card-media-right This class indicates that the media element is aligned to the right. 

<div class="tiadian-card tiadian-card-default">
    <div class="tiadian-card-media-top">
        <img src="" alt="">
    </div>
    <div class="tiadian-card-body"></div>
</div>

Horizontal alignment

The .tiadian-card-media-left or the .tiadian-card-media-right classes are used to reset border radius or similar where necessary. They don’t create the actual layout.

To do that, you could for example add the .tiadian-cover-container class from the Cover component. Add the tiadian-cover attribute to the image element and use the Grid and Width components to achieve the alignment. Create a <canvas> element with your image’s width and height, so that it will retain its dimensions, if the grid stacks on smaller viewports. This is just one way of creating a side by side layout.


<div class="tiadian-card tiadian-card-default tiadian-child-width-1-2" tiadian-grid>
    <div class="tiadian-card-media-left tiadian-cover-container">
        <img src="" alt="" tiadian-cover>
        <canvas width="" height=""></canvas>
    </div>
    <div>
        <div class="tiadian-card-body"></div>
    </div>
</div>

Badge

To position a badge inside a card, add the .tiadian-card-badge class to a container element. To style the badge, you can use one of the other components, for example the Label.


<div class="tiadian-card tiadian-card-body">
    <div class="tiadian-card-badge tiadian-label"></div>
</div>
