Close

<p class=“tiadian-text-lead”>Create a close icon that can be combined with different components.</p>

Usage

To apply this component, add the tiadian-close attribute to an <a> or <button> element.


<button type="button" tiadian-close></button>

<a href="" tiadian-close></a>


<button type="button" tiadian-close></button>

Large modifier

Add the .tiadian-close-large class for a larger close button.


<button class="tiadian-close-large" type="button" tiadian-close></button>


<button class="tiadian-close-large" type="button" tiadian-close></button>

Close in alerts

This is an example of how this component is used with an alert box from the Alert component.


<div tiadian-alert>
    <button class="tiadian-alert-close" type="button" tiadian-close></button>
</div>
