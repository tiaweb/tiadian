Column

<p class=“tiadian-text-lead”>Display your content in multiple text columns without having to split it in several containers.</p>

Usage

Add one of the .tiadian-column-* classes to an element to display its inline content in multiple columns.

Class Description
.tiadian-column-1-2 Display the content in two columns.
.tiadian-column-1-3 Display the content in three columns.
.tiadian-column-1-4 Display the content in four columns.
.tiadian-column-1-5 Display the content in five columns.
.tiadian-column-1-6 Display the content in six columns.

Responsive

TIAdian provides a number of responsive column classes. Basically they work just like the usual column classes, except that they have suffixes that represent the breakpoint from which they come to effect.

Class Description
.tiadian-column-1-2@s to .tiadian-column-1-6@s Only affects device widths of 640px and higher.
.tiadian-column-1-2@m to .tiadian-column-1-6@m Only affects device widths of 960px and higher.
.tiadian-column-1-2@l to .tiadian-column-1-6@l Only affects device widths of 1200px and higher.
.tiadian-column-1-2@xl to .tiadian-column-1-6@xl Only affects device widths of 1600px and higher.

Span all columns

To have an inline element span the whole width of the columns, add the .tiadian-column-span class to the inner element.

    ...
    <p class="tiadian-column-span"></p>
    ...
</div>
