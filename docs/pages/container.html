Container

<p class=“tiadian-text-lead”>This component allows you to align and center your page content.</p>

Usage

Add the .tiadian-container class to a block element to give it a max-width and wrap the main content of your website. The element will be centered and have padding on the sides, that adapts automatically for large screens.

Note The padding of nested containers will be removed to avoid unnecessary spacing.

<div class="tiadian-container"></div>

Size modifiers

Add one of the following classes to the container to apply a different max-width.

Class Description
.tiadian-container-xsmall Add this class for a xsmall container.
.tiadian-container-small Add this class for a small container.
.tiadian-container-large Add this class for a large container.
.tiadian-container-xlarge Add this class for a xlarge container.
.tiadian-container-expand Add this class, if you do not want to limit the container width but still want the dynamic horizontal padding. 
<div class="tiadian-container tiadian-container-xsmall"></div>

<div class="tiadian-container tiadian-container-small"></div>

<div class="tiadian-container tiadian-container-large"></div>

<div class="tiadian-container tiadian-container-xlarge"></div>

<div class="tiadian-container tiadian-container-expand"></div>

Container in sections

You can apply this component to modify the width of content inside sections from the Section component.

<div class="tiadian-section tiadian-section-primary">
    <div class="tiadian-container tiadian-container-small"></div>
</div>
