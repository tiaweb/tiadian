Description list

<p class=“tiadian-text-lead”>Easily create nice looking description lists, which come in different styles.</p>

Usage

Add the .tiadian-description-list class to display terms and descriptions below each other.


<dl class="tiadian-description-list">
    <dt></dt>
    <dd></dd>
</dl>
