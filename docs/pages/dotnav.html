Dotnav

<p class=“tiadian-text-lead”>Create a dot navigation to operate slideshows or to scroll to different page sections.</p>

Usage

To create a navigation with dots, use the following classes. This component is built with Flexbox. So to align a dotnav, you can use Flex component.

Class Description
.tiadian-dotnav Add this class to a <ul> element to define the Dotnav component. Use <a> elements as nav items within the list.
.tiadian-active Add this class to a list item to apply an active state. 
<ul class="tiadian-dotnav">
    <li class="tiadian-active"><a href=""></a></li>
    <li><a href=""></a></li>
</ul>

Vertical alignment

The dotnav can also be displayed vertically. Just add the .tiadian-dotnav-vertical modifier.

<ul class="tiadian-dotnav tiadian-dotnav-vertical">...</ul>

Position as overlay

To position the dotnav on top of an element or the Slideshow component for example, add one of the .tiadian-position-* classes from the Position component to a div element wrapping the dotnav. To create a position context on the container, add the .tiadian-position-relative class.

Use the .tiadian-light or .tiadian-dark class from the Inverse component to apply a light or dark color for better visibility.

<div class="tiadian-position-relative tiadian-light">

    <!-- The element which is wrapped goes here -->

    <div class="tiadian-position-bottom-center tiadian-position-small">
        <ul class="tiadian-dotnav">...</ul>
    </div>

</div>

Vertically center in viewport

The dotnav can also be centered vertically inside your viewport by adding the .tiadian-position-center-right and .tiadian-position-fixed classes from the Position component. This is useful for typical onepage websites.

<div class="tiadian-position-center-right tiadian-position-medium tiadian-position-fixed">
    <ul class="tiadian-dotnav tiadian-dotnav-vertical">...</ul>
</div>
