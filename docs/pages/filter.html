Filter

<p class=“tiadian-text-lead”>Filter or sort items in any given layout by meta data.</p>

The Filter component is often used together with the Grid component, especially the masonry grid, but it’s not bound to it. Any layout can be filtered or sorted no matter how the items are positioned. Items fade and move with smooth transitions between the different filtering and sorting states.

Usage

To apply this component, you need a container element with the tiadian-filter="target: SELECTOR" attribute. Inside this container create a list of filter controls as well as the layout items you want to filter. Use the target: SELECTOR option to select the element containing the layout items.


<div tiadian-filter="target: .js-filter">

    <!-- Filter controls -->
    <ul>
        <li><a href="#"></a></li>
    </ul>

    <!-- Layout items -->
    <ul class="js-filter">
        <li></li>
    </ul>

</div>

Next, we need to define the meta data for each layout item, for example which category the item belongs to. Use any HTML class or attribute to do so.

To apply a filter control, add the tiadian-filter-control attribute. To define the meta data that should be filtered, use the filter: SELECTOR option. The selector can be any CSS selector like a HTML class or an attribute you define for the layout items.


<div tiadian-filter="target: .js-filter">

    <ul>
        <li tiadian-filter-control="filter: .tag-blue"><a href="#"></a></li>
    </ul>

    <ul class="js-filter">
        <li class="tag-blue"></li>
    </ul>

</div>

If target is the only option in the tiadian-filter attribute value, you can also use tiadian-filter="SELECTOR". The same applies to the filter control. If filter is the only option in the tiadian-filter-control attribute value, you can also use tiadian-filter-control="SELECTOR".


<div tiadian-filter=".js-filter">

    <ul>
        <li tiadian-filter-control=".tag-blue"><a href="#"></a></li>
    </ul>

    <ul class="js-filter">
        <li class="tag-blue"></li>
    </ul>

</div>

The Filter component comes unstyled, which allows you to use any of the other TIAdian components to create the filter controls and layout items. For example, the Nav, Subnav and Tab components can be used to style the filter controls. Usually, the Grid component is used to create the item layout.

Meta Data

Items can have different meta data for filtering. You just need to define the HTML classes or data attributes and create the corresponding CSS selectors for the filter controls. This example uses data attributes for the filter instead of HTML classes.


<div tiadian-filter="target: .js-filter">

    <ul>
        <li tiadian-filter-control="[data-tags*='blue']"><a href="#"></a></li>
    </ul>

    <ul class="js-filter">
        <li data-tags="blue dark"></li>
    </ul>

</div>

Groups

To filter items by multiple criteria at the same time, the filter controls need to be grouped. Just add the group: NAME option to the tiadian-filter-control attribute and define group names for the meta data.


<div tiadian-filter="target: .js-filter">

    <ul>
        <li tiadian-filter-control="filter: [data-color='blue']; group: color"><a href="#"></a></li>
        <li tiadian-filter-control="filter: [data-color='white']; group: color"><a href="#"></a></li>
    </ul>

    <ul>
        <li tiadian-filter-control="filter: [data-size='small']; group: size"><a href="#"></a></li>
        <li tiadian-filter-control="filter: [data-size='large']; group: size"><a href="#"></a></li>
    </ul>

    <ul class="js-filter">
        <li data-color="blue" data-size="small"></li>

Sorting

To sort items alphanumerically, just add the sort: ATTRIBUTE option to the tiadian-filter-control attribute and define the meta data attribute by which the items should be sorted. By default, items are sorted in ascending order. Add the order: desc to sort items in descending order.


<div tiadian-filter="target: .js-filter">

    <ul>
        <li tiadian-filter-control="sort: data-color"><a href="#"></a></li>
        <li tiadian-filter-control="sort: data-color; order: desc"><a href="#"></a></li>
    </ul>

    <ul class="js-filter">
        <li data-color="blue"></li>
    </ul>

</div>

Component options

Any of these options can be applied to the component attribute. Separate multiple options with a semicolon. Learn more

Option Value Default Description
target String ‘’ The targeted list on which to apply the filter to.
selActive String, Boolean false A selector for the initially active filter controls.

target is the Primary option and its key may be omitted, if it’s the only option in the attribute value.

Learn more about JavaScript components.

TIAdian.filter(element, options);

Events

Name Description
beforeFilter Fires before the filter is applied.
afterFilter Fires after the filter has been applied.
