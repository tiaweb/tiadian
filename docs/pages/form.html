Form

<p class=“tiadian-text-lead”>Easily create nice looking forms with different styles and layouts.</p>

Usage

Add one of the following classes to form controls inside a <form> element to define them.

Class Description
.tiadian-input Add this class to <input> elements.
.tiadian-select Add this class to <select> elements.
.tiadian-textarea Add this class to <textarea> elements.
.tiadian-radio Add this class to <input type="radio"> elements to create radio buttons.
.tiadian-checkbox Add this class to <input type="checkbox"> elements to create checkboxes.
.tiadian-range Add this class to <input type="range"> elements to create range forms. 

<form>
    <select class="tiadian-select">
        <option></option>
        <option></option>
    </select>
    <textarea class="tiadian-textarea"></textarea>
    <input class="tiadian-radio" type="radio">
    <input class="tiadian-checkbox" type="checkbox">
    <input class="tiadian-range" type="range">
</form>

Add the .tiadian-fieldset class to a <fieldset> element and the .tiadian-legend class to a <legend> element to define a fieldset and a form legend.

Legend 

<form>
    <fieldset class="tiadian-fieldset">

        <legend class="tiadian-legend">Legend</legend>

        <div class="tiadian-margin">
            <input class="tiadian-input" type="text" placeholder="Input">
        </div>

        <div class="tiadian-margin">
            <select class="tiadian-select">
                <option>Option 01</option>
                <option>Option 02</option>
            </select>
        </div>

        <div class="tiadian-margin">
            <textarea class="tiadian-textarea" rows="5" placeholder="Textarea"></textarea>
        </div>

        <div class="tiadian-margin tiadian-grid-small tiadian-child-width-auto tiadian-grid">
            <label><input class="tiadian-radio" type="radio" name="radio2" checked> A</label>
            <label><input class="tiadian-radio" type="radio" name="radio2"> B</label>
        </div>

        <div class="tiadian-margin tiadian-grid-small tiadian-child-width-auto tiadian-grid">
            <label><input class="tiadian-checkbox" type="checkbox" checked> A</label>
            <label><input class="tiadian-checkbox" type="checkbox"> B</label>
        </div>

        <div class="tiadian-margin">
            <input class="tiadian-range" type="range" value="2" min="0" max="10" step="0.1">
        </div>

    </fieldset>
</form>

States modifiers

Provide the user with basic information through feedback states on form controls by using one of the following classes.

Class Description
.tiadian-form-danger Add this class to notify the user that the value is not validated.
.tiadian-form-success Add this class to notify the user that the value is validated.

Add the disabled attribute to a form control and it will be appear muted.


<div class="tiadian-margin">
    <input class="tiadian-input tiadian-form-danger tiadian-form-width-medium" type="text" placeholder="form-danger" value="form-danger">
</div>

<div class="tiadian-margin">
    <input class="tiadian-input tiadian-form-success tiadian-form-width-medium" type="text" placeholder="form-success" value="form-success">
</div>

<div class="tiadian-margin">
    <input class="tiadian-input tiadian-form-width-medium" type="text" placeholder="disabled" value="disabled" disabled>
</div>

Size modifiers

Add one of the classes to an <input>, <select> or <textarea> element to modify its size.

Class Description
.tiadian-form-large Add this class to make the element larger.
.tiadian-form-small Add this class to make the element smaller. 

<form>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-medium tiadian-form-large" type="text" placeholder="Large">
    </div>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-medium" type="text" placeholder="Default">
    </div>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-medium tiadian-form-small" type="text" placeholder="Small">
    </div>

</form>

Width modifiers

Add one of the following classes to an <input>, <select> or <textarea> element to adjust its width.

Class Description
.tiadian-form-width-large Applies a width of 500px.
.tiadian-form-width-medium Applies a width of 200px.
.tiadian-form-width-small Applies a width of 130px.
.tiadian-form-width-xsmall Applies a width of 40px. 

<form>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-large" type="text" placeholder="Large">
    </div>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-medium" type="text" placeholder="Medium">
    </div>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-small" type="text" placeholder="Small">
    </div>

    <div class="tiadian-margin">
        <input class="tiadian-input tiadian-form-width-xsmall" type="text" placeholder="X-Small">
    </div>

</form>

You can also apply the .tiadian-width-* classes from the Width component to form controls.


<form>
    <input class="tiadian-input tiadian-width-1-2" type="text" placeholder="tiadian-width-1-2">
</form>

Blank modifier

Add the .tiadian-form-blank class to minimize the styling of form controls.


<form>
    <input class="tiadian-input tiadian-form-blank tiadian-form-width-medium" type="text" placeholder="Form blank">
</form>

Layout

Define labels and controls and apply a stacked or horizontal layout to form elements. Layout modifiers can be added to any parent element like the <fieldset> element. This makes it possible to have different form layouts for each fieldset.

Class Description
.tiadian-form-stacked Add this class to display labels on top of controls.
.tiadian-form-horizontal Add this class to display labels and controls side by side.
.tiadian-form-label Add this class to define form labels.
.tiadian-form-controls Add this class to define form controls. 

<form class="tiadian-form-stacked">
    <div>
        <label class="tiadian-form-label"></label>
        <div class="tiadian-form-controls">...</div>
    </div>
    <div>
        <div class="tiadian-form-label"></div>
        <div class="tiadian-form-controls">...</div>
    </div>
</form>
Radio


<form class="tiadian-form-stacked">

    <div class="tiadian-margin">
        <label class="tiadian-form-label" for="form-stacked-text">Text</label>
        <div class="tiadian-form-controls">
            <input class="tiadian-input" id="form-stacked-text" type="text" placeholder="Some text...">
        </div>
    </div>

    <div class="tiadian-margin">
        <label class="tiadian-form-label" for="form-stacked-select">Select</label>
        <div class="tiadian-form-controls">
            <select class="tiadian-select" id="form-stacked-select">
                <option>Option 01</option>
                <option>Option 02</option>
            </select>
        </div>
    </div>

    <div class="tiadian-margin">
        <div class="tiadian-form-label">Radio</div>
        <div class="tiadian-form-controls">
            <label><input class="tiadian-radio" type="radio" name="radio1"> Option 01</label><br>
            <label><input class="tiadian-radio" type="radio" name="radio1"> Option 02</label>
        </div>
    </div>

</form>

Form and icons

You use an icon from the Icon component inside a form. Add the .tiadian-form-icon class to a <span> element. Group it with an <input> element by adding the .tiadian-inline class from the Utility component to a container element around both. The icon has to come first in the markup. By default, the icon will be placed on the left side of the form. To change the alignment, add the .tiadian-form-icon-flip class.


<div class="tiadian-inline">
    <span class="tiadian-form-icon" tiadian-icon="icon: user"></span>
    <input class="tiadian-input">
</div>
Radio


<form>

    <div class="tiadian-margin">
        <div class="tiadian-inline">
            <span class="tiadian-form-icon" tiadian-icon="icon: user"></span>
            <input class="tiadian-input" type="text">

    </div>

    <div class="tiadian-margin">
        <div class="tiadian-inline">
            <span class="tiadian-form-icon tiadian-form-icon-flip" tiadian-icon="icon: lock"></span>
            <input class="tiadian-input" type="text">
        </div>
    </div>

</form>

Custom controls

To replace a file inputs or select forms with your own HTML content, like a button or text, add the tiadian-form-custom attribute to a container element.

File

Use a button or text as a file input.


<div tiadian-form-custom>
    <input type="file">
    <button type="button"></button>
</div>
Here is a text
upload 

<form>

    <div class="tiadian-margin">
        <div tiadian-form-custom>
            <input type="file">
            <button class="tiadian-button tiadian-button-default" type="button" tabindex="-1">Select</button>
        </div>
    </div>

    <div class="tiadian-margin">
        <span class="tiadian-text-middle">Here is a text</span>
        <div tiadian-form-custom>
            <input type="file">
            <span class="tiadian-link">upload</span>
        </div>
    </div>

    <div class="tiadian-margin" tiadian-margin>
        <div tiadian-form-custom="target: true">
            <input type="file">
            <input class="tiadian-input tiadian-form-width-medium" type="text" placeholder="Select file" disabled>
        </div>
        <button class="tiadian-button tiadian-button-default">Submit</button>
    </div>

</form>

Select

Use a button, text or a link as a select form. Just add the target: SELECTOR option to the tiadian-form-custom attribute to select where the option value should be displayed. target: true will select the adjacent element in the markup.


<div tiadian-form-custom="target: true">
    <select>
        <option></option>
        <option></option>
    </select>
    <button type="button"></button>
</div>


<form>

        <div tiadian-form-custom="target: true">

                <option value="4">Option 04</option>
