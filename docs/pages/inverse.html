Inverse

Inverse the style of any component for light or dark backgrounds.

Usage

Add the .tiadian-light class to improve the visibility of objects on dark backgrounds in a light style. When using a dark style, add the .tiadian-dark class to elements on a light background.

Note Only one class will come to effect, depending on the style you are using. For example, when using a style with a light background, you can only apply the .tiadian-light class to optimize content on dark backgrounds, as text will already be displayed in a dark color on light backgrounds â€” and vice versa.

Customize using Less

The Inverse component includes additional styles to implement the flexible inverse behaviour. If your project does not make use of these styles, you can leave disable the Inverse component when compiling Less.
