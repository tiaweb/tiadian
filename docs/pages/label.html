Label

Indicate important notes and highlight parts of your content.

Usage

To create a label, just add the .tiadian-label class to a <span> element.

      <span class="tiadian-label"></span>
Default 
      <span class="tiadian-label">Default</span>

Style modifiers

Add one of the following classes for additional styles.

Class Description
.tiadian-label-success Indicates success or a positive message.
.tiadian-label-warning Indicates a warning.
.tiadian-label-danger Indicates an error or important message. 
      <span class="tiadian-label tiadian-label-success"></span>
Default Success Warning Danger 
      <span class="tiadian-label">Default</span>
