Link

Define different styles to integrate links into specific content.

Muted modifier

The Base component determines the default link color. If you want the link to a apply a muted style instead, just add the .tiadian-link-muted class to the anchor element. You can also add the class to a parent element and it will be applied to all <a> elements inside it.

      <a class="tiadian-link-muted" href="#"></a>

Text modifier

To make a link appear like body text and apply a hover effect, add the .tiadian-link-text class to the anchor element. You can also add the class to a parent element and it will be applied to all <a> elements inside it. This is useful for link lists in the page footer.

      <a class="tiadian-link-text" href="#"></a>

    
      <ul class="tiadian-list tiadian-link-text">
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
</ul>

Heading modifier

To style a link inside a heading element, add the .tiadian-link-heading class to the anchor element.

      <h3><a class="tiadian-link-heading" href="#"></a></h3>

Heading

      <h3><a class="tiadian-link-heading" href="#">Heading</a></h3>

Reset modifier

To reset a link's color, so that it inherits the color from its parent, add the .tiadian-link-reset class. There won't be any hover effect at all. This is useful for links inside heading elements. You can also add the class to a parent element and it will be applied to all <a> elements inside it.

Toggle

To use an anchor as parent element and apply the link style on one of its child elements, just add the .tiadian-link-toggle class to the parent element and one of the .tiadian-link-* classes to the child element. For instance, you can link the whole card and still have the hover effect on the heading.

      <a class="tiadian-link-toggle" href="#">
    <span class="tiadian-link-heading"></span>
</a>
