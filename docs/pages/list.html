List

Easily create nice looking lists, which come in different styles.

Usage

To apply this component, add the .tiadian-list class to an unordered or ordered list. The list will now display without any spacing or list-style.

      <ul class="tiadian-list">
    <li></li>
    <li></li>
    <li></li>
</ul>
  • List item 1
  • List item 2
  • List item 3
      <ul class="tiadian-list">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

Bullet modifier

Add the .tiadian-list-bullet class to display bullets in front of the list items.

      <ul class="tiadian-list tiadian-list-bullet">...</ul>
  • List item 1
  • List item 2
  • List item 3
      <ul class="tiadian-list tiadian-list-bullet">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

Divider modifier

Add the .tiadian-list-divider class to separate list items with lines.

      <ul class="tiadian-list tiadian-list-divider">...</ul>
  • List item 1
  • List item 2
  • List item 3
      <ul class="tiadian-list tiadian-list-divider">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

Striped modifier

Add zebra-striping to a list using the .tiadian-list-striped class.

      <ul class="tiadian-list tiadian-list-striped">...</ul>
  • List item 1
  • List item 2
  • List item 3
      <ul class="tiadian-list tiadian-list-striped">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

Large modifier

Add the .tiadian-list-large class to increase the spacing between list items. You can combine this with any of the other list modifiers.

      <ul class="tiadian-list tiadian-list-large">...</ul>
