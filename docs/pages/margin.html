Margin

A collection of utility classes to add spacing between elements.

Usage

Add one or more of the following classes to any element to create the same vertical and/or horizontal margin that a paragraph usually has.

Class Description
.tiadian-margin Adds top margin, if it is preceded by another element, and always bottom margin.
.tiadian-margin-top Adds top margin.
.tiadian-margin-bottom Adds bottom margin.
.tiadian-margin-left Adds left margin.
.tiadian-margin-right Adds right margin.

Remove margin

Add one of the following classes to remove margin from block elements.

Class Description
.tiadian-margin-remove Removes all margins.
.tiadian-margin-remove-top Removes top margin.
.tiadian-margin-remove-bottom Removes bottom margin.
.tiadian-margin-remove-left Removes left margin.
.tiadian-margin-remove-right Removes right margin.
.tiadian-margin-remove-vertical Removes all vertical margins.
.tiadian-margin-remove-adjacent Removes the top margin of the directly succeeding element.
.tiadian-margin-remove-first-child Removes the top margin of the first child element.
.tiadian-margin-remove-last-child Removes the bottom margin of the last child element. 
<h1 class="tiadian-margin-remove"></h1>

Responsive

TIAdian provides a number of responsive classes to remove margin. Basically, they work just like the usual margin remove classes, except that they have suffixes that represent the breakpoint from which they come to effect.

Class Description
.tiadian-margin-remove-left@s
.tiadian-margin-remove-right@s		 Affects device widths of 640px and larger.
.tiadian-margin-remove-left@m
.tiadian-margin-remove-right@m		 Affects device widths of 960px and larger.
.tiadian-margin-remove-left@l
.tiadian-margin-remove-right@l		 Affects device widths of 1200px and larger.
.tiadian-margin-remove-left@xl
.tiadian-margin-remove-right@xl		 Affects device widths of 1600px and larger.

Auto margin

Add one of the following classes to set auto margin. This can be useful to center or otherwise align block elements with a fixed width as well as flex elements.

Class Description
.tiadian-margin-auto Sets left and right margin to auto, horizontally centering block and flex elements.
.tiadian-margin-auto-top Sets top margin to auto, pushing block and flex elements to the bottom.
.tiadian-margin-auto-bottom Sets bottom margin to auto, pushing block and flex elements to the top.
.tiadian-margin-auto-left Sets left margin to auto, pushing block and flex elements to the right.
.tiadian-margin-auto-right Sets right margin to auto, pushing block and flex elements to the left
.tiadian-margin-auto-vertical Sets top and bottom margin to auto, vertically centering only flex elements.

Dynamic wrapping

To add spacing to stacking elements, for example inline elements that wrap on smaller viewports, just add the tiadian-margin attribute to their parent container. It will automatically add the .tiadian-margin-small-top class to the lower element.

<div tiadian-margin>
    <button></button>
    <button></button>
</div>

Component options

Any of these options can be applied to the component attribute. Separate multiple options with a semicolon. Learn more

Option Value Default Description
margin String tiadian-margin-small-top This class is added to items that break into the next row, typically to create margin to the previous row.
first-column String tiadian-first-column This class is added to the first element in each row.
