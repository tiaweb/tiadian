Modal

Create modal dialogs with different styles and transitions.

Usage

The Modal component consists of an overlay, a dialog and an optional close button. You can use any element to toggle a modal dialog. To enable the necessary JavaScript, add the tiadian-toggle attribute. An <a> element needs to be linked to the modal's id. If you are using another element, like a button, just add the tiadian-toggle="target: #ID" attribute to target the id of the modal container.

Add the tiadian-modal attribute to a <div> element to create the modal container and an overlay that blanks out the page. It is important to add an id to indicate the element for toggling. Use the following classes to define the modal's sections.

Class Description
.tiadian-modal-dialog Add this class to a child <div> element to create the dialog
.tiadian-modal-body Add this class to create padding between the modal and its content.
.tiadian-modal-title Add this class to a heading element to create the modal title.
.tiadian-modal-close Add this class to an <a> or <button> element to create a close button and enable its functionality. 
      <!-- This is a button toggling the modal -->
<button tiadian-toggle="target: #my-id" type="button"></button>

<!-- This is the modal -->
<div id="my-id" tiadian-modal>
    <div class="tiadian-modal-dialog tiadian-modal-body">
        <h2 class="tiadian-modal-title"></h2>
        <button class="tiadian-modal-close" type="button"></button>
    </div>
</div>

Close button

To create a close button, enable its functionality and add proper styling and positioning, add the .tiadian-modal-close-default class to an <a> or <button> element. To place the close button outside the modal, add the .tiadian-modal-close-outside class.

Add the tiadian-close attribute from the Close component, to apply a close icon.

      <div id="my-id">
    <div class="tiadian-modal-dialog">
        <button class="tiadian-modal-close-default" type="button" tiadian-close></button>
    </div>
</div>

<div id="my-id">
    <div class="tiadian-modal-dialog">
        <button class="tiadian-modal-close-outside" type="button" tiadian-close></button>
    </div>
</div>

Full modifier

To create a modal, that fills the entire page, add the .tiadian-modal-full class. It is also recommended to add the .tiadian-modal-close-full class to the close button, so that it adapts its styling.

      <div id="my-id" class="tiadian-modal-full" tiadian-modal>
    <div class="tiadian-modal-dialog">
        <button class="tiadian-modal-close-full tiadian-close-large" type="button" tiadian-close></button>
    </div>
</div>

By default, the page will scroll with the modal, if its content exceeds the window height. To apply a scrollbar inside the modal, add the tiadian-overflow-auto attribute from the Utility component to the modal body.

      <div id="my-id" tiadian-modal>
    <div class="tiadian-modal-dialog" tiadian-overflow-auto></div>
</div>

        <iframe src="" tiadian-video></iframe>
    </div>

Component options

Any of these options can be applied to the component attribute. Separate multiple options with a semicolon. Learn more

Option Value Default Description
esc-close Boolean true Close the modal when the Esc key is pressed.
bg-close Boolean true Close the modal when the background is clicked.
stack Boolean false Stack modals, when more than one is open. By default, the previous modal will be hidden.
container String true Define a target container via a selector to specify where the modal should be appended in the DOM. Setting it to false will prevent this behavior.
cls-page String 'tiadian-modal-page' Class to add to <body> when modal is active
cls-panel String 'tiadian-modal-dialog' Class of the element to be considered the panel of the modal
sel-close String '.tiadian-modal-close, .tiadian-modal-close-default, .tiadian-modal-close-outside, .tiadian-modal-close-full' CSS selector for all elements that should trigger the closing of the modal

Modal dialogs

The component comes with a number of prepared modal dialogs that you can use for user interaction. You can call the dialog directly from JavaScript and use callback functions to process the user input.

Code Description
TIAdian.modal.alert('TIAdian alert!') Show an alert box with one button.
TIAdian.modal.confirm('TIAdian confirm!') Show a confirm dialog with your message and two buttons.
TIAdian.modal.prompt('Name:', 'Your name') Show a dialog asking for a text input.
TIAdian.modal.dialog('<p>TIAdian dialog!</p>'); Show dialog with any HTML content.

To process the user input, the modal uses a promise based interface which provides a then() function to register your callback functions. The TIAdian.modal.dialog function however will return the modal itself.

      TIAdian.modal.confirm('TIAdian confirm!').then(function() {
    console.log('Confirmed.')
}, function () {
    console.log('Rejected.')
});

The returned promise has a property dialog, which holds a reference to the modal itself. This lets you manipulate e.g. the markup of the modal's element.

      const modal = TIAdian.modal.confirm('TIAdian confirm!').dialog; // The modal component
const el = modal.$el; // The modal element
