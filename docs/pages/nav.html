Nav

Defines different styles for list navigations.

Usage

To apply this component, use the following classes.

Class Description
.tiadian-nav Add this class to a <ul> element to define the Nav component. Use <a> elements as nav items within the list.
.tiadian-active Add this class to a list item to apply an active state to a menu item. 
      <ul class="tiadian-nav">
    <li class="tiadian-active"><a href=""></a></li>
    <li><a href=""></a></li>
</ul>
  • Active
  • Item
  • Item
      <div class="tiadian-width-1-2@s tiadian-width-2-5@m">
    <ul class="tiadian-nav tiadian-nav-default">
        <li class="tiadian-active"><a href="#">Active</a></li>
        <li><a href="#">Item</a></li>
        <li><a href="#">Item</a></li>
    </ul>
</div>

Note By default, the nav has no styling. That's why it is important to add a modifier class. In our example we are using the .tiadian-nav-default class.

Accordion

By default, child menu items are always visible. To apply an accordion effect, just add the tiadian-nav attribute to the main <ul>. Add the .tiadian-nav-parent-icon class to apply icons, indicating parent items.

Note The attribute automatically sets the .tiadian-nav class, so you don't have to apply it manually.

Component options

Any of these options can be applied to the component attribute. Separate multiple options with a semicolon. Learn more

Option Value Default Description
targets CSS selector > .tiadian-parent The element(s) to toggle.
toggle CSS selector > a The toggle element(s).
content CSS selector > ul The content element(s).
collapsible Boolean true Allow all items to be closed.
multiple Boolean false Allow multiple open items.
transition String ease The transition to use.
animation String true The space separated names of animations to use. Comma separate for animation out.
duration Number 200 The animation duration in milliseconds.

JavaScript

Learn more about JavaScript components.

Initialization

      TIAdian.nav(element, options);

Methods

The following methods are available for the component:

Toggle

      TIAdian.nav(element).toggle(index, animate);

Toggles the content pane.

Name Type Default Description
index String, Integer, Node 0 Nav pane to toggle. 0 based index.
animate Boolean true Suppress opening animation by passing false.
