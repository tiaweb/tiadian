Navbar

Create a navigation bar that can be used for your main site navigation.

Usage

The Navbar component consists of a navbar container, the navbar itself and one or more navigations.

Element Description
tiadian-navbar Add this attribute to a <nav> element to define the Navbar component.
.tiadian-navbar-container Add this class to the same <nav> element or a parent element to add the navbar background style.
.tiadian-navbar-left
.tiadian-navbar-center
.tiadian-navbar-right		 Add one of these classes to a <div> element to align the navigation.
.tiadian-navbar-nav Add this class to a <ul> element to create the navigation. Use <a> elements as menu items within the list.
.tiadian-parent Add this class to indicate a parent menu item.
.tiadian-active Add this class to indicate an active menu item. 
<nav class="tiadian-navbar-container" tiadian-navbar>
    <div class="tiadian-navbar-left">
        <ul class="tiadian-navbar-nav">
            <li class="tiadian-active"><a href=""></a></li>
            <li class="tiadian-parent"><a href=""></a></li>
            <li><a href=""></a></li>
        </ul>
    </div>
</nav>

    <div class="tiadian-navbar-left">...</div>
    <div class="tiadian-navbar-center">...</div>
    <div class="tiadian-navbar-right">...</div>
</nav>

Click mode

A parent item inside the navbar can be enabled by either hovering or clicking the toggle. Just add the mode: click option to the tiadian-navbar attribute.

<nav class="tiadian-navbar-container" tiadian-navbar="mode: click">...</nav>

Transparent modifier

When using an image or colored background for the hero section of your website, you might want to turn the navbar transparent. Just add the .tiadian-navbar-transparent class to the <nav> element. If necessary, add the .tiadian-light or .tiadian-dark class from the Inverse component to adjust the navbar's color.

<nav class="tiadian-navbar-container tiadian-navbar-transparent" tiadian-navbar>...</nav>

Content item

You can also add custom content to the navbar, like text, icons, buttons or forms. Add the .tiadian-navbar-item class to a <div> element that serves as a container for your content

<div class="tiadian-navbar-container" tiadian-navbar>
    <div class="tiadian-navbar-left">
        <a href="" class="tiadian-navbar-item tiadian-logo"></a>
        <ul class="tiadian-navbar-nav">...</ul>
        <div class="tiadian-navbar-item">...</div>
    </div>
</div>

Add the .tiadian-logo class from the Utility component to an <a> or <div> element to indicate your brand.

Centered logo

You can create a split menu with a centered logo. Just add the tiadian-navbar-center-left class to one navbar and the tiadian-navbar-center-right class to another within the same navbar container. This will keep your logo centered and align the menus to the left and right.

Note An extra div element is required to support IE 11.

<div class="tiadian-navbar-container" tiadian-navbar>
    <div class="tiadian-navbar-center">
        <div class="tiadian-navbar-center-left"><div>...</div></div>
        <a href="" class="tiadian-navbar-item tiadian-logo"></a>
        <div class="tiadian-navbar-center-right"><div>...</div></div>
    </div>
</div>

Toggle item

Add the .tiadian-navbar-toggle class and the tiadian-navbar-toggle-icon attribute to an <a> or <div> element to create an icon as a toggle. By default, there is no JavaScript attached to the toggle. As an example, you can add an off-canvas navigation from the Off-canvas component or a modal from the Modal component.

<div class="tiadian-navbar-container" tiadian-navbar>
    <div class="tiadian-navbar-left">
        <a class="tiadian-navbar-toggle" tiadian-navbar-toggle-icon href=""></a>
    </div>
</div>
Menu 
<nav class="tiadian-navbar tiadian-navbar-container tiadian-margin">

    </div>

Dropdowns

A navbar can contain a dropdown from the Dropdown component. Just add the .tiadian-navbar-dropdown modifier to the dropdown, so it fits perfectly into the navbar's styling. Add the .tiadian-navbar-dropdown-nav class to navs inside the dropdown.

        <a href=""></a>
        <div class="tiadian-navbar-dropdown">
            <ul class="tiadian-nav tiadian-navbar-dropdown-nav">...</ul>

Class Description
.tiadian-navbar-dropdown-width-2 Add this class to double the dropdown's width.
.tiadian-navbar-dropdown-width-3 Add this class to triple the dropdown's width.
.tiadian-navbar-dropdown-width-4 Add this class to multiply the dropdown's width by four.
.tiadian-navbar-dropdown-width-5 Add this class to multiply the dropdown's width by five.

Boundary alignment

Dropdowns can be aligned to the navbar's boundary. Just append the boundary-align: true parameter to the tiadian-navbar attribute. Add the align: left, align: center or align: right option to change the alignment. By default, dropdowns are aligned to the left.

<nav class="tiadian-navbar-container" tiadian-navbar="boundary-align: true; align: center;">...</nav>

Dropbar

Sticky Navbar

For a basic sticky navbar, wrap the navbar inside a container with the tiadian-sticky attribute from the Sticky component.

The navbar itself has a modifier class tiadian-navbar-sticky that ensures an optimized styling for the sticky state (for example an additional box shadow). To let the sticky component dynamically add and remove that class, set cls-active: tiadian-navbar-sticky. To ensure that the class is added to the navbar container, set sel-target: .tiadian-navbar-container.

Note You can view more examples in the tests for the Sticky Navbar.

<div tiadian-sticky="sel-target: .tiadian-navbar-container; cls-active: tiadian-navbar-sticky">
    <nav class="tiadian-navbar-container" tiadian-navbar>...</nav>

Transparent Sticky Navbar

When you use a transparent navbar, your markup contains the .tiadian-navbar-transparent class together with .tiadian-light or .tiadian-dark. When the navbar is sticky, it is usually required to remove these classes and add them when the navbar returns to the non-sticky state. To do that, set cls-inactive: tiadian-navbar-transparent tiadian-light.

By default, the immediate toggling of classes does not look ideal. Instead, we can set top: 200 to let the navbar disappear and then re-appear when the user has scrolled 200px past the navbar. In that case we can also define to have the navbar slide in with an animation, just set animation: tiadian-animation-slide-top.

<div tiadian-sticky="animation: tiadian-animation-slide-top; sel-target: .tiadian-navbar-container; cls-active: tiadian-navbar-sticky; cls-inactive: tiadian-navbar-transparent tiadian-light; top: 200">
    <nav class="tiadian-navbar-container" tiadian-navbar>...</nav>
</div>

Component options

Any of these options can be applied to the component attribute. Separate multiple options with a semicolon. Learn more

Option Value Default Description
align String left Dropdown alignment: left, right, center
mode String click, hover Comma separated list of dropdown trigger behavior modes: click, hover
delay-show Number 0 Delay time in hover mode before a dropdown is shown in milliseconds.
delay-hide Number 800 Delay time in hover mode before a dropdown is hidden in milliseconds.
boundary CSS selector window Referenced element to keep the dropdown's visibility.
boundary-align Boolean false Align the dropdown to the boundary.
offset Number 0 The offset of the dropdown container.
dropbar Boolean false Enable or disable dropbar behavior.
dropbar-mode String slide The mode in which the dropbar appears: slide, push
duration Number 200 The dropbar transition duration.

Initialization

Events

The following events will be triggered on elements with this component attached:

Name Description
beforeshow Fires before an item is shown. Can prevent showing by calling preventDefault() on the event.
show Fires after an item is shown.
shown Fires after the item's show animation has completed.
beforehide Fires before an item is hidden. Can prevent hiding by calling preventDefault() on the event.
hide Fires after an item's hide animation has started.
hidden Fires after an item is hidden.
