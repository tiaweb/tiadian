Overlay

Create an image overlay, which comes in different styles.

Usage

Add the .tiadian-overlay class to an element following an image to create the overlay panel. To create a position context, add the .tiadian-inline class from the Utility component to a container element around both. Finally, add one of the .tiadian-position-* classes from the Position component to align the overlay.

      <div class="tiadian-inline">
    <img src="" alt="">
    <div class="tiadian-overlay tiadian-position-bottom"></div>
</div>

Note Add the .tiadian-light or .tiadian-dark class, so that elements will be optimized for better visibility on dark or light images.

Default Lorem ipsum dolor sit amet, consectetur adipiscing elit.

      <div class="tiadian-inline">
    <img src="images/photo.jpg" alt="">
    <div class="tiadian-overlay tiadian-light tiadian-position-bottom">
        <p>Default Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
    </div>
</div>

Style modifiers

By default, the overlay has padding, but no additional styling. Add one of these modifier classes to add a background color to the overlay.

For a simple background color, add the .tiadian-overlay-default class.

Overlay icon

To display an overlay icon, add the tiadian-overlay-icon attribute to a <span> element inside the overlay.

      <div class="tiadian-position-center">
    <span tiadian-overlay-icon></span>
