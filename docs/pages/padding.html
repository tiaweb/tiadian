Padding

A collection of utility classes to add spacing between elements and their content.

Usage

To apply this component, add one of the following classes to a block element.

Class Description
.tiadian-padding Adds default padding to the element.
.tiadian-padding-small Adds small padding to the element.
.tiadian-padding-large Adds large padding to the element. 
      <div class="tiadian-padding"></div>

Remove padding

In some cases you want to remove padding from an element, rather than adding it. To do so, apply one of the following classes.

Class Description
.tiadian-padding-remove Removes all padding from an element.
.tiadian-padding-remove-top Removes top padding from an element.
.tiadian-padding-remove-bottom Removes bottom padding from an element.
.tiadian-padding-remove-left Removes left padding from an element.
.tiadian-padding-remove-right Removes right padding from an element.
.tiadian-padding-remove-vertical Removes top and bottom padding from an element.
.tiadian-padding-remove-horizontal Removes left and right padding from an element.
