Pagination

Easily create a nice looking pagination to navigate through pages.

Usage

The Pagination component consists of button-like styled links, that are aligned side by side in a horizontal list.

Class Description
.tiadian-pagination Add this class to a <ul> element to define the Pagination component. Use <a> elements as pagination items within the list.
.tiadian-active Add this class to a list item to apply an active state and use a <span> instead of an <a> element.
.tiadian-disabled Add this class to a list item to apply a disabled state and use a <span> instead of an <a> element.

Previous and next

To apply a previous and next button, add the tiadian-pagination-previous or tiadian-pagination-next attribute to a <span> element inside a pagination item.

      <ul class="tiadian-pagination">
    <li><a href=""><span tiadian-pagination-previous></span></a></li>
    <li><a href=""><span tiadian-pagination-next></span></a></li>
</ul>
  • Previous
  • Next 
      <ul class="tiadian-pagination">
    <li><a href="#"><span class="tiadian-margin-small-right" tiadian-pagination-previous></span> Previous</a></li>
    <li class="tiadian-margin-auto-left"><a href="#">Next <span class="tiadian-margin-small-left" tiadian-pagination-next></span></a></li>
</ul>
