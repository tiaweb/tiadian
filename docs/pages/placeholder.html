Placeholder

Create a placeholder space that can be used for uploading files via drag and drop.

Usage

This component applies a specially styled area to create a placeholder space, which can be useful, for example in combination with the Upload component. Just add the .tiadian-placeholder class to a <div> element.

      <div class="tiadian-placeholder"></div>
