Position

A collection of utility classes to position content.

Usage

To apply this component, add one of the .tiadian-position-* classes to a block element. When using this component to place content on top of an image, add the .tiadian-inline class from the Utility component to a container element around the image and the element to create a position context.

Note This is often used to position an overlay from the Overlay component. Use the .tiadian-light or .tiadian-dark class from the Inverse component to apply a light or dark color for better visibility.

Class Description
.tiadian-position-top Positions the element at the top.
.tiadian-position-left Positions the element at the left.
.tiadian-position-right Positions the element at the right.
.tiadian-position-bottom Positions the element at the bottom.

    <!-- Place any content, like an image, here -->

Outside

There are two classes to center elements outside on the left and right of the parent container. This is useful to position the Slidenav component outside of an Slideshow or Slider component.

Class Description
.tiadian-position-center-left-out Positions the element vertically centered outside on the left.
.tiadian-position-center-right-out Positions the element vertically centered outside on the right.

Note Once the outside positioned element sticks out of the viewport to the right, it will cause a horizontal scrollbar. Use the Visibility component to hide the outside positioned element on a smaller viewports and show an inside positioned element instead.

      <div class="tiadian-position-center-left-out"></div>

Utility classes

This component features a number of general position utility classes:

Class Description
.tiadian-position-relative Add this class to apply relative positioning.
.tiadian-position-absolute Add this class to apply absolute positioning.
.tiadian-position-fixed Add this class to apply fixed positioning.
.tiadian-position-z-index Add this class to apply a z-index of 1.
