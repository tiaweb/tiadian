Search

Easily create a nice looking search.

Usage

The Search component consists of a search form and the search input itself.

Class/Attribute Description
.tiadian-search Add this class to a container element to define the Search component.
.tiadian-search-input Add this class to an <input> element to create the search field. 
      <form class="tiadian-search">
    <input class="tiadian-search-input" type="search" placeholder="">
</form>

By default, the search has no additional styling. In this example we are using the .tiadian-search-default modifier.

      <form class="tiadian-search tiadian-search-default">
    <input class="tiadian-search-input" type="search" placeholder="Search...">
</form>

Search icon

To create a search icon, add the tiadian-search-icon attribute to a <span> element. To change the alignment, add the .tiadian-search-icon-flip class.

      <form class="tiadian-search tiadian-search-default">
    <span tiadian-search-icon></span>
    <input class="tiadian-search-input" type="search" placeholder="">
</form>

    
      <div class="tiadian-margin">
    <form class="tiadian-search tiadian-search-default">
        <span tiadian-search-icon></span>
        <input class="tiadian-search-input" type="search" placeholder="Search...">
    </form>
</div>

<div class="tiadian-margin">
    <form class="tiadian-search tiadian-search-default">
        <span class="tiadian-search-icon-flip" tiadian-search-icon></span>
        <input class="tiadian-search-input" type="search" placeholder="Search...">
    </form>
</div>

Clickable

To enable an action, use an <a> or <button> element to create the icon.

      <form class="tiadian-search tiadian-search-default">
    <a href="" tiadian-search-icon></a>
    <input class="tiadian-search-input" type="search" placeholder="">
</form>

    
      <div class="tiadian-margin">
    <form class="tiadian-search tiadian-search-default">
        <a href="" tiadian-search-icon></a>
        <input class="tiadian-search-input" type="search" placeholder="Search...">
    </form>
</div>

<div class="tiadian-margin">
    <form class="tiadian-search tiadian-search-default">
        <a href="" class="tiadian-search-icon-flip" tiadian-search-icon></a>
        <input class="tiadian-search-input" type="search" placeholder="Search...">
    </form>
</div>

Default modifier

To apply the default search styling, add the .tiadian-search-default class.

      <form class="tiadian-search tiadian-search-default">...</form>

    
      <form class="tiadian-search tiadian-search-default">
    <span tiadian-search-icon></span>
    <input class="tiadian-search-input" type="search" placeholder="Search...">
</form>

Large modifier

To increase the size of the search, for example when creating an overlay search, add the .tiadian-search-large class.

      <form class="tiadian-search tiadian-search-large">...</form>

    
      <form class="tiadian-search tiadian-search-large">
    <span tiadian-search-icon></span>
    <input class="tiadian-search-input" type="search" placeholder="Search...">
</form>

Navbar modifier

A search can be used inside a navbar from the Navbar component. Just add the .tiadian-search-navbar class.

      <form class="tiadian-search tiadian-search-navbar">...</form>

    
      <nav class="tiadian-navbar-container" tiadian-navbar>
    <div class="tiadian-navbar-left">

        <div class="tiadian-navbar-item">
            <form class="tiadian-search tiadian-search-navbar">
                <span tiadian-search-icon></span>
                <input class="tiadian-search-input" type="search" placeholder="Search...">
            </form>
        </div>

    </div>
</nav>

Toggle

To apply a hover state to the search icon when using an <a> or <button> element, add the .tiadian-search-toggle class. To create the search icon itself, add the tiadian-search-icon attribute.

      <a class="tiadian-search-toggle" href="" tiadian-search-icon></a>

    
      <a class="tiadian-search-toggle" href="" tiadian-search-icon></a>

Search in navbar

The search icon can be used as a toggle to open the search inside a navbar – as an overlay, drop or dropdown – or outside in a modal. Just add the .tiadian-navbar-toggle class to the icon.

      <div class="tiadian-navbar-container" tiadian-navbar>
    <div class="tiadian-navbar-left">
        <a class="tiadian-navbar-toggle" tiadian-search-icon tiadian-toggle href=""></a>
    </div>
</div>
