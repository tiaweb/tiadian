Section

Create horizontal layout sections with different background colors and styles.

Usage

Sections are used to separate the content of a page into differently styled blocks. To apply this component, just add the .tiadian-section class to a container element. You can place a container from the Container component to modify the width of the content inside sections and add horizontal padding. Note that the padding of a nested container will be reset.

      <div class="tiadian-section">
    <div class="tiadian-container"></div>
</div>

By default, a section is blank. That is why it is important to add a modifier class for styling. In our example we are using the .tiadian-section-muted class. Normally, the section is the outermost element of a page, so this example doesn't quite reflect the realistic usage of this component.

Style modifiers

To apply different background colors and paddings, add one of the following classes.

Class Description
.tiadian-section-default Adds the default background color of your site.
.tiadian-section-muted Adds a muted background color.
.tiadian-section-primary Adds a primary background color.
.tiadian-section-secondary Adds a secondary background color. 
      <div class="tiadian-section tiadian-section-primary"></div>

Preserve color

The .tiadian-section-primary and .tiadian-section-secondary classes are extending the inverse style from the Inverse component automatically. If you want to prevent this behavior, for example because you are using cards in these sections, add the .tiadian-preserve-color class.

      <div class="tiadian-section tiadian-section-primary tiadian-preserve-color"></div>

Size modifier

You can add different paddings to each section or remove it altogether.

Class Description
.tiadian-section-xsmall Add this class to decrease a section's padding to a minimum.
.tiadian-section-small Add this class to decrease a section's padding.
.tiadian-section-large Add this class to increase a section's padding.
.tiadian-section-xlarge Add this class to further increase a section's padding.
.tiadian-padding-remove-vertical Add this class from the Padding component to remove a section's padding.

Overlap modifier

Some TIAdian themes use richer styles, including textured backgrounds and borders. In this case, you can add the .tiadian-section-overlap class, to apply a border image and a negative offset to the following section. Note that this only works in styles that implement the feature, if there are subsequent sections.

      <div class="tiadian-section tiadian-section-overlap"></div>
