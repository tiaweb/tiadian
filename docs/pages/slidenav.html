Slidenav

Defines a navigation with previous and next buttons to flip through slideshows.

Usage

To create a navigation with previous and next buttons, just add the tiadian-slidenav attribute to <a> elements. Add the previous or next parameter to the attribute to style the nav items as previous or next buttons.

      <a href="" tiadian-slidenav-next></a>
<a href="" tiadian-slidenav-previous></a>

    
      <a href="#" tiadian-slidenav-previous></a>
<a href="#" tiadian-slidenav-next></a>

Large modifier

To increase the size of the slidenav icons, add the .tiadian-slidenav-large class.

      <a href="" class="tiadian-slidenav-large" tiadian-slidenav-next></a>
<a href="" class="tiadian-slidenav-large" tiadian-slidenav-previous></a>

    
      <a href="#" class="tiadian-slidenav-large" tiadian-slidenav-previous></a>
<a href="#" class="tiadian-slidenav-large" tiadian-slidenav-next></a>

Slidenav container

To display a conjoint slidenav, wrap the slidenav items inside a <div> element and add the .tiadian-slidenav-container class, as well as one of the .tiadian-position-* classes.

      <div class="tiadian-slidenav-container">
    <a href="" tiadian-slidenav-previous></a>
    <a href="" tiadian-slidenav-next></a>
</div>

    
      <div class="tiadian-slidenav-container">
    <a href="" tiadian-slidenav-previous></a>
    <a href="" tiadian-slidenav-next></a>
</div>

Position as overlay

To position the slidenav on top of an element or the Slideshow component for example, just add one of the .tiadian-position-* classes from the Position component. To create a position context on the container, add the .tiadian-position-relative class.

Use the .tiadian-light or .tiadian-dark class from the Inverse component to apply a light or dark color for better visibility.

      <div class="tiadian-position-relative tiadian-light">

    <!-- The element which is wrapped goes here -->

    <a class="tiadian-position-center-left" href="" tiadian-slidenav-previous></a>
    <a class="tiadian-position-center-right" href="" tiadian-slidenav-next></a>

</div>

Note You can also apply the Visibility component, so that the slidenav only appears on hover.

      <div class="tiadian-position-relative tiadian-visible-toggle tiadian-light" tabindex="-1" tiadian-slideshow>

    <ul class="tiadian-slideshow-items">
        <li>
            <img src="images/photo.jpg" alt="" tiadian-cover>
        </li>
        <li>
            <img src="images/dark.jpg" alt="" tiadian-cover>
        </li>
        <li>
            <img src="images/light.jpg" alt="" tiadian-cover>
        </li>
    </ul>

    <a class="tiadian-slidenav-large tiadian-position-center-left tiadian-position-small tiadian-hidden-hover" href="#" tiadian-slidenav-previous tiadian-slideshow-item="previous"></a>
    <a class="tiadian-slidenav-large tiadian-position-center-right tiadian-position-small tiadian-hidden-hover" href="#" tiadian-slidenav-next tiadian-slideshow-item="next"></a>

</div>
