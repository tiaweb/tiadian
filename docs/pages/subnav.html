Subnav

Defines different styles for a sub navigation.

Usage

To apply this component, use the following classes. To align a subnav, for example to horizontally center it, you can use the Flex component.

Class Description
.tiadian-subnav Add this class to a <ul> element to define the Subnav component. Use <a> elements as nav items within the list.
.tiadian-active Add this class to a list item to apply an active state.

To add list items without a link, use a <span> element instead of an <a>. Alternatively, disable an <a> element by adding the .tiadian-disabled class to the <li> element and remove the href attribute from the anchor to make it inaccessible through keyboard navigation.

      <ul class="tiadian-subnav">
    <li class="tiadian-active"><a href=""></a></li>
    <li><a href=""></a></li>
    <li><span></span></li>
</ul>
  • Active
  • Item
  • Item
  • Disabled
      <ul class="tiadian-subnav" tiadian-margin>
    <li class="tiadian-active"><a href="#">Active</a></li>
    <li><a href="#">Item</a></li>
    <li><a href="#">Item</a></li>
    <li><span>Disabled</span></li>
</ul>

Note For a better layout, if items should wrap into the next row, add the tiadian-margin attribute from the Margin component.

Divider modifier

Add the .tiadian-subnav-divider class to separate menu items with lines.

      <ul class="tiadian-subnav tiadian-subnav-divider">...</ul>
  • Active
  • Item
  • Item
      <ul class="tiadian-subnav tiadian-subnav-divider" tiadian-margin>
    <li class="tiadian-active"><a href="#">Active</a></li>
    <li><a href="#">Item</a></li>
    <li><a href="#">Item</a></li>
</ul>

Pill modifier

Add the .tiadian-subnav-pill class to highlight the active menu item with a background.

      <ul class="tiadian-subnav tiadian-subnav-pill">...</ul>
  • Active
  • Item
  • Item
      <ul class="tiadian-subnav tiadian-subnav-pill" tiadian-margin>
    <li class="tiadian-active"><a href="#">Active</a></li>
    <li><a href="#">Item</a></li>
    <li><a href="#">Item</a></li>
</ul>

Subnav with Dropdown

You can also use a dropdown from the Dropdown component with a subnav.

      <ul class="tiadian-subnav">
    <li>
        <a href=""></a>
        <div tiadian-dropdown="mode: click;"></div>
    </li>
</ul>
  • Active
  • Item
  • More
    • Active
    • Item
    • Header
    • Item
    • Item
    • Item
      <ul class="tiadian-subnav tiadian-subnav-pill" tiadian-margin>
    <li class="tiadian-active"><a href="#">Active</a></li>
    <li><a href="#">Item</a></li>
    <li>
        <a href="#">More <span tiadian-icon="icon:  triangle-down"></span></a>
        <div tiadian-dropdown="mode: click;">

                <li class="tiadian-active"><a href="#">Active</a></li>

                <li class="tiadian-nav-divider"></li>
                <li><a href="#">Item</a></li>
            </ul>
        </div>
    </li>
</ul>
