Thumbnav

Create a flexible thumbnail navigation.

Usage

To create a navigation with thumbnails, use the following classes. This component is built with Flexbox. So to align a thumbnav, you can use Flex component.

Class Description
.tiadian-thumbnav Add this class to a <ul> element to define the Thumbnav component. Nest your thumbnail images inside <a> elements within the list.
.tiadian-active Add this class to a list item to apply an active state. 
      <ul class="tiadian-thumbnav">
    <li class="tiadian-active"><a href=""><img src="" alt=""></a></li>
    <li><a href=""><img src="" alt=""></a></li>
</ul>

Note For a better layout, if items should wrap into the next row, add the tiadian-margin attribute from the Margin component.

      <ul class="tiadian-thumbnav" tiadian-margin>
    <li class="tiadian-active"><a href="#"><img src="images/photo.jpg" width="100" alt=""></a></li>
    <li><a href="#"><img src="images/dark.jpg" width="100" alt=""></a></li>
    <li><a href="#"><img src="images/light.jpg" width="100" alt=""></a></li>
</ul>

Vertical alignment

The thumbnav can also be displayed vertically. Just add the .tiadian-thumbnav-vertical modifier.

      <ul class="tiadian-thumbnav tiadian-thumbnav-vertical">...</ul>

    
      <ul class="tiadian-thumbnav tiadian-thumbnav-vertical" tiadian-margin>
    <li class="tiadian-active"><a href="#"><img src="images/photo.jpg" width="100" alt=""></a></li>
    <li><a href="#"><img src="images/dark.jpg" width="100" alt=""></a></li>
    <li><a href="#"><img src="images/light.jpg" width="100" alt=""></a></li>
</ul>

Position as overlay

To position the thumbnav on top of an element or the Slideshow component for example, add one of the .tiadian-position-* classes from the Position component to a div element wrapping the thumbnav. To create a position context on the container, add the .tiadian-position-relative class.

      <div class="tiadian-position-relative">

    <!-- The element which is wrapped goes here -->

    <div class="tiadian-position-bottom-center tiadian-position-small">
        <ul class="tiadian-thumbnav">...</ul>
    </div>

</div>

    
      <div class="tiadian-position-relative" tiadian-slideshow="animation: fade">

    <ul class="tiadian-slideshow-items">
        <li>
            <img src="images/photo.jpg" alt="" tiadian-cover>
        </li>
        <li>
            <img src="images/dark.jpg" alt="" tiadian-cover>
        </li>
        <li>
            <img src="images/light.jpg" alt="" tiadian-cover>
        </li>
    </ul>

    <div class="tiadian-position-bottom-center tiadian-position-small">
        <ul class="tiadian-thumbnav">
            <li tiadian-slideshow-item="0"><a href="#"><img src="images/photo.jpg" width="100" alt=""></a></li>
            <li tiadian-slideshow-item="1"><a href="#"><img src="images/dark.jpg" width="100" alt=""></a></li>
            <li tiadian-slideshow-item="2"><a href="#"><img src="images/light.jpg" width="100" alt=""></a></li>
        </ul>
    </div>

</div>
