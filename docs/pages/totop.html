Totop

Create a simple to-top scroller.

Usage

The Totop component allows you to apply an icon to your to-top scroller. Just add the tiadian-totop attribute to an <a> element.

To add a smooth effect when using the totop to jump up the page, use the Scroll component.

      <a href="" tiadian-totop></a>

    
      <a href="#" tiadian-totop tiadian-scroll></a>
