Panel

TIAdian uses panels to outline certain sections of your content. These can be arranged in grid columns from the Grid component component, for example.

Add the .tiadian-panel class to a <div> element to create a position context, set box-sizing to border-box, apply a clearfix and to remove the bottom margin of its last child element.

Clearing and floating

Floating elements are taken from the document flow and aligned to the left or right side of their container. It is important to clear floats or in the worst case, you might end up with a scrambled site. The following classes will help you to set up basic layouts.

Class Description
.tiadian-float-left Add this class to float the element to the left.
.tiadian-float-right Add this class to float the element to the right.
.tiadian-clearfix Add this class to a parent container to clear floats. Alternatively, you can create a new block format context, e.g. by adding the .tiadian-overflow-hidden class.
Right
Left
      <div class="tiadian-clearfix">
    <div class="tiadian-float-right">
        <div class="tiadian-card tiadian-card-default tiadian-card-body">Right</div>
    </div>
    <div class="tiadian-float-left">
        <div class="tiadian-card tiadian-card-default tiadian-card-body">Left</div>
    </div>
</div>

Responsive objects

In TIAdian <img>, <canvas>, <audio> and <video> elements adapt to the width of their parent container by default. To apply responsive behavior to iframes, add the tiadian-responsive attribute . For other elements or to apply a different behavior, just add one of the following classes.

Class Description
.tiadian-responsive-width Add this class to apply the same responsive behavior to any other element. It adjusts the object's width according to its parent's width, keeping the original aspect ratio.
.tiadian-responsive-height Add this class to adjust the object's height (instead of its width) according to its parent's height, keeping the original aspect ratio.
.tiadian-preserve-width Add this class to avoid the default responsive behavior and preserve the original image dimensions. You can also add the class to a parent element and it will be applied to all relevant elements content it. If you are embedding Google Maps into your site, you may need this to fix the map's images. 
      <img class="tiadian-responsive-height" src="" alt="">

<iframe src="" width="" height="" frameborder="0" tiadian-responsive></iframe>

Class Description

.tiadian-transform-origin-center-left The transition originates from the left.
.tiadian-transform-origin-center-right The transition originates from the right.
.tiadian-transform-origin-bottom-left The transition originates from the bottom left.
.tiadian-transform-origin-bottom-center The transition originates from the bottom.
