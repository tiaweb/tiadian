Visibility

Use responsive visibility classes to display or hide elements on different devices.

Usage

Add one of these classes to hide an element.

Class Description
.tiadian-hidden Hides the element on any device. This is more of a legacy class. The recommended way to do this, is to use the hidden attribute.
.tiadian-invisible Hides the element without removing it from the document flow. 
      <div hidden></div>

<div class="tiadian-invisible"></div>

Responsive

This component provides responsive classes to hide or display elements on different viewports.

Hidden

Add one of the .tiadian-hidden-* classes to hide the element from screens larger than a specified width.

Class Description
tiadian-hidden@s Only affects device widths of 640px and larger.
tiadian-hidden@m Only affects device widths of 960px and larger.
tiadian-hidden@l Only affects device widths of 1200px and larger.
tiadian-hidden@xl Only affects device widths of 1600px and larger. 
      <!-- Hidden on tablets and larger -->
<div class="tiadian-hidden@m"></div>

Note In this example the green elements are hidden on screens that are larger than the defined breakpoint. Resize your browser window to see the effect.

Toggle

To display elements on hover or focus only, add the .tiadian-visible-toggle class to a parent element and one of the following classes to any child elements which should be hidden.

Class Description
.tiadian-hidden-hover The element is removed from the document flow when hidden.
.tiadian-invisible-hover The element is not removed from the document flow when hidden.

The child elements will be displayed when the parent element is hovered or focused. Add tabindex="0" to the parent element to allow it to receive focus through keyboard navigation and on touch devices.

If there are a or button elements within the hidden child element, they are already focusable through keyboard navigation and will make the child element appear. Therefore, add tabindex="-1" so the parent element is still focusable on touch devices.

      <div class="tiadian-visible-toggle" tabindex="0">
    <div class="tiadian-hidden-hover"></div>
</div>

Touch

Add the .tiadian-hidden-touch class to hide elements on touch devices and the .tiadian-hidden-notouch to hide elements on devices without a touch screen.

      <!-- Hidden on touch devices -->
<div class="tiadian-hidden-touch"></div>

<!-- Hidden on no-touch devices -->
<div class="tiadian-hidden-notouch"></div>
