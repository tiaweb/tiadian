Width

Define the width of elements for different viewport sizes.

TIAdian's Width component is often used in combination with grids to split content into responsive columns. You can apply fractions, automatic width or expand units to fill the remaining space and combine these modes.

Usage

Add one of the .tiadian-width-* classes to an element to determine its size. Typically, you would use a grid from the Grid component and its child elements to create the units.

Class Description
.tiadian-width-1-1 Fills 100% of the available width.
.tiadian-width-1-2 The element takes up halves of its parent container.
.tiadian-width-1-3 to .tiadian-width-2-3 The element takes up thirds of its parent container.
.tiadian-width-1-4 to .tiadian-width-3-4 The element takes up fourths of its parent container.
.tiadian-width-1-5 to .tiadian-width-4-5 The element takes up fifths of its parent container.
.tiadian-width-1-6 to .tiadian-width-5-6 The element takes up sixths of its parent container.

Note We remove redundancy into each set of unit classes, so that for instance instead of .tiadian-width-3-6 you should use .tiadian-width-1-2.

Auto & expand

The Width component provides additional modifiers to give you more flexibility in the distribution of items.

Class Description
.tiadian-width-auto The item expands only to the width of its own content.
.tiadian-width-expand The item expands to fill up the remaining space of the grid container. 
<div tiadian-grid>
    <div class="tiadian-width-auto"></div>
    <div class="tiadian-width-expand"></div>
</div>

    <div class="tiadian-width-auto">

    <div class="tiadian-width-expand">
        <div class="tiadian-card tiadian-card-default tiadian-card-body">Expand</div>
    </div>

Equal child widths

To create a grid whose child elements' widths are evenly split, you don't have to apply the same class to each list item within the grid. Just add one of the .tiadian-child-width-* classes to the grid itself.

Class Description
.tiadian-child-width-1-2 All elements take up half of their parent container.
.tiadian-child-width-1-3 All elements take up a third of their parent container.
.tiadian-child-width-1-4 All elements take up a fourth of their parent container.
.tiadian-child-width-1-5 All elements take up a fifth of their parent container.
.tiadian-child-width-1-6 All elements take up a sixth of their parent container.
.tiadian-child-width-auto Divides the grid into equal units depending on the content size.
.tiadian-child-width-expand Divides the grid into equal units depending on the available space.

Fixed width

In addition to the calculated width classes, you can also add one of the following classes, which apply fixed widths.

Class Description
.tiadian-width-small Applies a fixed width of 150px.
.tiadian-width-medium Applies a fixed width of 300px.
.tiadian-width-large Applies a fixed width of 450px.
.tiadian-width-xlarge Applies a fixed width of 600px.
.tiadian-width-xxlarge Applies a fixed width of 750px.

Mixing widths

You can also combine .tiadian-child-width-* classes with .tiadian-width-* classes for individual items. That way it is possible, for example, to create a grid with one item that has a specific width and all other items expanding to fill the remaining space.

Responsive width

TIAdian provides a number of responsive widths classes. Basically they work just like the usual width classes, except that they have suffixes that represent the breakpoint from which they come to effect. These classes can be combined with the Visibility component. This is great to adjust your layout and content for different device sizes.

Class Description
.tiadian-width-*
.tiadian-child-width-*		 Affects all device widths, grid columns stay side by side.
.tiadian-width-*@s
.tiadian-child-width-*@s		 Affects device widths of 640px and larger. Grid columns will stack on smaller sizes.
.tiadian-width-*@m
.tiadian-child-width-*@m		 Affects device widths of 960px and larger. Grid columns will stack on smaller sizes.
.tiadian-width-*@l
.tiadian-child-width-*@l		 Affects device widths of 1200px and larger. Grid columns will stack on smaller sizes.
.tiadian-width-*@xl
.tiadian-child-width-*@xl		 Affects device widths of 1600px and larger. Grid columns will stack on smaller sizes.
