Cdn host Tiadian
Pengantar
Mulailah dengan Tiaweb kerangka kerja untuk membangun situs mobile-responsif

Mulai cepat
Ingin menambah Tiaweb dengan cepat ke proyek Anda? Gunakan Tiaweb cdn disediakan secara gratis

CSS
Salin-tempel stylesheet <link> ke dalam Anda <head> sebelum semua stylesheet lainnya untuk memuat CSS kami.

<!-- TIAWEB CSS -->
    <link rel="stylesheet" href="https://tiaweb.gitlab.io/tiadian/style.css">
JS
Banyak komponen kami membutuhkan penggunaan JavaScript agar berfungsi. Secara khusus, Tempatkan Javascript Tiaweb berikut <script> di dekat akhir halaman Anda, tepat sebelum tag penutup , untuk mengaktifkannya. 
<!-- Tiaweb JavaScript -->
 <script src="https://tiaweb.gitlab.io/tiadian/script.js"></script>
<script src="https://tiaweb.gitlab.io/tiadian/icons.js"></script>
Template pemula
Pastikan halaman Anda diatur dengan standar desain dan pengembangan terbaru. Itu berarti menggunakan doctype HTML5 dan termasuk meta tag viewport untuk perilaku responsif yang tepat. Gabungkan semuanya dan halaman Anda akan terlihat seperti ini: 
<!doctype html>
<html lang="en">
  <head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <!-- TIAWEB CSS -->
    <link rel="stylesheet" href="https://tiaweb.gitlab.io/tiadian/style.css">
       <title>Hello, world!</title>
  </head>
  <body>
    <h1>Hello, world!</h1>

    <!-- Tiaweb JavaScript -->
     <script src="https://tiaweb.gitlab.io/tiadian/script.js"></script>
<script src="https://tiaweb.gitlab.io/tiadian/icons.js"></script>
   

</body>
</html>
